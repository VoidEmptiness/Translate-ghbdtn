# Translate-ghbdtn

Обновленная версия простого переводчика для преобразования текста между раскладками QWERTY и ЙЦУКЕН.

Исходная версия: https://github.com/VoidsTeam/Translate_wasd-

## Зачем это нужно

Этот переводчик был создан для моего друга, который часто забывает переключить раскладку клавиатуры и пишет по-русски на английской раскладке. В планах - реализовать автоматический перевод такого текста в Telegram без участия пользователя.

## Запуск

1. Установите Python 3.
2. Скопируйте файлы
   ```
   git clone https://github.com/VoidEmptiness/Translate-ghbdtn.git
   ```
3. Для PyQt5 версии установите зависимости:
   ```
   pip install -r PyQt5
   ```
4. Запустите нужную версию:
   ```
   python translator_tkinter.py    # Версия на Tkinter
   python translator_pyqt5.py      # Версия на PyQt5
   ```

## Файл соответствий

Символы для перевода берутся из файла `base.txt` в формате:
```
q-й
w-ц
e-у
...
```
